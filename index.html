Tidy some data

Here we tidy the data,

my_data <- spanish %>%
  select(., x = X14461gll, y = X14473gll) %>%
  filter(., y < 0.06, x < 0.075) 

my_data  %>%
  ggplot(., aes(x= x , y = y )) +
  geom_point() +
  geom_smooth(method = 'lm')
